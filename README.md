# 股神盯盘 - 桌面股票监控小工具

一个简洁美观的A股实时监控桌面小工具，基于 PyQt5 开发。无边框半透明窗口，可拖动，支持系统托盘。

## 功能特点

- **实时监控** - 实时显示股票价格、涨跌幅，红涨绿跌
- **桌面悬浮** - 无边框半透明窗口，可拖动到屏幕任意位置
- **股票管理** - 支持按代码/名称搜索股票，一键添加/删除
- **置顶功能** - 重要股票可置顶，显示图标和灰色背景
- **拖拽排序** - 在管理界面拖动调整股票显示顺序
- **分时走势** - 点击股票查看当日真实分时走势图、均价线、成交量
- **K线图表** - 支持日K、周K、月K，带MA5/10/20/60均线和成交量MA线
- **技术指标** - MACD、KDJ、RSI 三大指标，深色专业主题
- **做T计算器** - 快速计算做T盈亏，实时显示收益金额和百分比
- **ETF支持** - 支持沪深ETF基金（如513120、159611等）
- **系统托盘** - 关闭窗口后最小化到托盘，右键可退出
- **自动保存** - 股票列表、置顶状态、窗口设置自动保存到配置文件

## 使用方法

### 启动程序

```bash
python stock_widget.py
```

### 窗口操作

| 操作 | 说明 |
|------|------|
| 拖动窗口 | 按住窗口任意位置拖动 |
| 💰 按钮 | 打开做T计算器 |
| ⚙ 按钮 | 打开股票管理对话框 |
| 🔧 按钮 | 打开设置（透明度、刷新间隔） |
| × 按钮 | 关闭窗口（最小化到托盘） |
| 点击股票 | 查看股票详情（分时/K线/指标） |
| 滚轮 | 上下滚动查看更多股票 |

### 股票管理

点击 **⚙** 按钮打开管理界面：

1. **添加股票** - 输入代码或名称搜索，双击或点击"添加选中"
2. **删除股票** - 选中后点击"删除选中"
3. **置顶股票** - 选中后点击"置顶"，置顶股票显示📌图标，排在最前
4. **拖拽排序** - 直接拖动调整顺序，点击"确定"保存

### 股票详情

点击主窗口中的股票，打开详情对话框：

| 图表类型 | 说明 |
|---------|------|
| 分时 | 当日真实分钟级行情走势、均价线、昨收参考线、成交量 |
| 日K | 日K线蜡烛图，MA5/10/20/60均线，成交量含MA线 |
| 周K | 周K线蜡烛图，均线和成交量 |
| 月K | 月K线蜡烛图，均线和成交量 |
| MACD | DIF、DEA双线 + MACD红绿柱状图 |
| KDJ | K、D、J三线 + 超买超卖区域 + J值柱状图 |
| RSI | RSI(6)曲线 + 80/20超买超卖参考线 + 区域填充 |

图表支持鼠标悬停查看详细数据，支持工具栏缩放和平移。

### 做T计算器

点击 **💰** 按钮，输入买入价、卖出价、手数，实时显示盈亏金额和百分比。

## 配置文件

程序自动在当前目录生成 `config.json`：

```json
{
  "stocks": [
    "sh000001",
    "600519",
    "513120",
    "159611"
  ],
  "pinned": [
    "sh000001"
  ],
  "opacity": 0.85,
  "refresh_interval": 5
}
```

| 字段 | 说明 |
|------|------|
| `stocks` | 监控的股票代码列表 |
| `pinned` | 置顶的股票代码列表 |
| `opacity` | 窗口透明度（0.5-1.0） |
| `refresh_interval` | 数据刷新间隔（秒） |

## 支持的证券类型

| 代码开头 | 类型 | 示例 |
|---------|------|------|
| 6 | 沪市主板 | 600519 贵州茅台 |
| 5 | 沪市ETF | 513120 半导体ETF |
| 0 | 深市主板 | 000001 平安银行 |
| 3 | 创业板 | 300454 深信服 |
| 1 | 深市ETF | 159611 食品ETF |
| sh000xxx | 沪市指数 | sh000001 上证指数 |

## 数据来源

| 数据类型 | API来源 |
|---------|---------|
| 实时行情 | 腾讯财经 (`qt.gtimg.cn`) |
| 分时走势 | 腾讯分钟行情 (`web.ifzq.gtimg.cn`) |
| K线数据 | 腾讯K线API（前复权） |
| 股票搜索 | 新浪财经 (`suggest3.sinajs.cn`) |

## 常见问题

**Q: 窗口消失了？**
A: 点击系统托盘图标重新显示。

**Q: 搜不到某只股票？**
A: 确保输入6位数字代码或正确名称，目前仅支持A股和ETF。

**Q: 如何退出程序？**
A: 右键托盘图标选择"退出"，或在启动终端按 Ctrl+C。

**Q: 数据不更新？**
A: 可能是网络问题或非交易时间。分时/K线数据有5分钟缓存，可切换图表类型刷新。

**Q: ETF代码无法识别？**
A: 确保5开头ETF（沪市）和1开头ETF（深市）代码正确，程序已支持自动识别。
