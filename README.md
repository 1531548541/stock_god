# 股神盯盘 - 桌面股票监控小工具

一个简洁美观的A股股票实时监控桌面小工具，采用 PyQt5 开发，无边框透明窗口，可拖动，支持系统托盘。

## 功能特点

- **实时监控** - 实时显示股票价格、涨跌幅
- **桌面小工具** - 无边框透明窗口，可拖动到屏幕任意位置
- **股票管理** - 支持按代码/名称搜索股票，一键添加/删除
- **拖拽排序** - 在管理界面拖动调整股票显示顺序
- **滚动支持** - 股票较多时支持上下滚动查看
- **系统托盘** - 关闭窗口后最小化到托盘，右键可退出
- **自动保存** - 股票列表自动保存，下次启动自动加载

## 安装

### 1. 克隆或下载项目

```bash
cd stock_god
```

### 2. 安装依赖

```bash
pip install PyQt5 requests
```

或使用 requirements.txt：

```bash
pip install -r requirements.txt
```

## 使用方法

### 启动程序

```bash
python stock_widget.py
```

### 窗口操作

| 操作 | 说明 |
|------|------|
| 拖动窗口 | 按住窗口任意位置拖动 |
| ⚙ 按钮 | 打开股票管理对话框 |
| × 按钮 | 关闭窗口（最小化到托盘） |
| 滚轮 | 上下滚动查看更多股票 |

### 股票管理

点击 **⚙** 按钮打开管理界面：

1. **添加股票**
   - 在搜索框输入股票代码（6位数字）或股票名称
   - 点击"🔍 搜索"或按回车
   - 选中搜索结果，点击"➕ 添加选中"或双击添加

2. **删除股票**
   - 在"当前监控的股票"列表中选中要删除的股票
   - 点击"➖ 删除选中"

3. **拖拽排序**
   - 直接用鼠标拖动股票项调整顺序
   - 点击"✓ 确定"保存更改

4. **保存配置**
   - 点击"✓ 确定"保存所有更改并刷新首页
   - 点击"✗ 取消"放弃更改

## 配置文件

程序会自动在当前目录生成 `config.json` 配置文件：

```json
{
  "stocks": [
    "600519",
    "000001",
    "600036"
  ]
}
```

手动编辑此文件也可添加股票，每行一个6位股票代码。

## 支持的股票

- **上海交易所** - 6开头（如 600519 贵州茅台）
- **深圳交易所** - 0开头（如 000001 平安银行）
- **科创板** - 688开头
- **创业板** - 300开头

## 数据说明

- 数据来源：腾讯财经API
- 刷新频率：5秒
- 颜色显示：红色表示上涨，绿色表示下跌

## 注意事项

- 本程序仅供学习交流使用

## 常见问题

**Q: 窗口消失到哪里了？**

A: 点击托盘图标可以重新显示窗口。

**Q: 搜索功能搜不到股票？**

A: 请确保输入的是6位数字代码或正确的股票名称，目前仅支持A股。

**Q: 如何退出程序？**

A: 右键点击托盘图标，选择"退出"。

**Q: 数据不更新？**

A: 可能是网络问题或非交易时间，请在交易时间内使用。
