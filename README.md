# 股神盯盘 - 桌面股票监控小工具

一个简洁美观的A股股票实时监控桌面小工具，采用 PyQt5 开发，无边框透明窗口，可拖动，支持系统托盘。

## 功能特点

- **实时监控** - 实时显示股票价格、涨跌幅
- **桌面小工具** - 无边框透明窗口，可拖动到屏幕任意位置
- **股票管理** - 支持按代码/名称搜索股票，一键添加/删除
- **置顶功能** - 重要股票可置顶，显示图标和灰色背景
- **拖拽排序** - 在管理界面拖动调整股票显示顺序
- **股票详情** - 点击股票查看分时图、K线图、技术指标
- **做T计算器** - 辅助计算做T收益，支持历史记录
- **滚动支持** - 股票较多时支持上下滚动查看
- **系统托盘** - 关闭窗口后最小化到托盘，右键可退出
- **自动保存** - 股票列表、置顶状态、设置自动保存

## 使用方法

### 启动程序

```bash
python stock_widget.py
```

### 窗口操作

| 操作 | 说明 |
|------|------|
| 拖动窗口 | 按住窗口任意位置拖动 |
| 💰 按钮 | 打开做T计算器 |
| ⚙ 按钮 | 打开股票管理对话框 |
| × 按钮 | 关闭窗口（最小化到托盘） |
| 点击股票 | 查看股票详情（分时图/K线图） |
| 滚轮 | 上下滚动查看更多股票 |

### 股票管理

点击 **⚙** 按钮打开管理界面：

1. **添加股票**
   - 在搜索框输入股票代码（6位数字）或股票名称
   - 点击"🔍 搜索"或按回车
   - 选中搜索结果，点击"➕ 添加选中"或双击添加
   - 新增股票会自动插入到置顶股票之后

2. **删除股票**
   - 在"当前监控的股票"列表中选中要删除的股票
   - 点击"➖ 删除选中"

3. **置顶股票**
   - 选中要置顶的股票
   - 点击"📌 置顶"按钮切换置顶状态
   - 置顶股票显示 📌 图标和灰色背景
   - 再次点击可取消置顶

4. **拖拽排序**
   - 直接用鼠标拖动股票项调整顺序
   - 点击"✓ 确定"保存更改

### 股票详情

点击主窗口中的股票，打开详情对话框：
- **分时图** - 显示当日分时走势和成交量
- **K线图** - 显示日K线、均线
- **技术指标** - MACD、KDJ等指标

### 做T计算器

点击 **💰** 按钮打开做T计算器：
- 输入买入价、卖出价、股数
- 自动计算做T收益
- 支持保存历史记录

### 设置

在主窗口右键菜单选择"设置"：
- **窗口透明度** - 调整窗口透明度（50%-100%）
- **刷新间隔** - 设置数据刷新频率（1-60秒）

## 配置文件

程序会自动在当前目录生成 `config.json` 配置文件：

```json
{
  "stocks": [
    "sh000001",
    "600519",
    "000001"
  ],
  "pinned": [
    "sh000001"
  ],
  "opacity": 0.85,
  "refresh_interval": 5
}
```

配置说明：
- `stocks`: 监控的股票代码列表
- `pinned`: 置顶的股票代码列表
- `opacity`: 窗口透明度（0.5-1.0）
- `refresh_interval`: 刷新间隔（秒）

## 支持的股票

- **上海交易所** - 6开头（如 600519 贵州茅台）
- **深圳交易所** - 0开头（如 000001 平安银行）
- **科创板** - 688开头
- **创业板** - 300开头
- **指数** - 支持上证指数（sh000001）等主流指数

## 数据说明

- 数据来源：腾讯财经API、新浪财经API
- 刷新频率：默认5秒，可在设置中调整
- 颜色显示：红色表示上涨，绿色表示下跌

## 常见问题

**Q: 窗口消失到哪里了？**

A: 点击托盘图标可以重新显示窗口。

**Q: 搜索功能搜不到股票？**

A: 请确保输入的是6位数字代码或正确的股票名称，目前仅支持A股。

**Q: 如何退出程序？**

A: 右键点击托盘图标，选择"退出"；或按 Ctrl+C。

**Q: 数据不更新？**

A: 可能是网络问题或非交易时间，请在交易时间内使用。

**Q: 置顶的股票为什么不显示在最上面？**

A: 确保config.json中pinned字段包含该股票代码，重新启动程序即可。
